# 📝MERN-blog 소개

> JavaScript를 활용한 풀스택 개발 프로젝트<br/>

### 상세 기능

- 로그인/회원가입✅
- 게시글 작성✅, 수정✅, 삭제✅
- 댓글 작성, 수정, 삭제✅
- 좋아요 기능✅

<br>

| 🛠 FRONTEND | 📡 BACKEND |
|--------------|-------------|
| Vite         | Node.js     |
| React.js     | Express.js  |
| TypeScript   | MongoDB Atlas |
| SCSS         | mongoose    |
|              | JWT         |
|              | helmet      |
